# Contributing

Thanks for helping improve `truckersmp-cli-community`.

## Project goals

This fork aims to:

- keep TruckersMP usable on Linux/Proton
- stay compatible with upstream `truckersmp-cli` where possible
- merge useful fixes with attribution
- improve diagnostics and reliability
- avoid breaking existing user configs

## Branch naming

Use clear branch names:

- `fix/...` for bug fixes
- `feature/...` for new features
- `docs/...` for documentation
- `ci/...` for GitHub Actions or tooling

Examples:

- `fix/steamcmd-libraryfolders`
- `feature/doctor-command`
- `docs/install-guide`

## Pull requests

Before opening a PR:

- test the CLI locally
- explain what changed
- mention which distro and Steam setup you tested
- update documentation if user-facing behavior changed

## Compatibility

Avoid breaking existing commands or config files unless there is a strong reason.
